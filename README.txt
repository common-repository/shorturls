== What is ShortUrlsEZ? ==

ShortUrlsEZ.com is a newly launched Israel based link shortener service, it pays to shorten and share urls. Like other url shorteners ShortUrlsEZ also provides some good features to all its users, like good customer support, good referral program, quick signup option, multiple tools, daily payments etc.

== How many ShortUrlsEZ Pay ==

look into [https://ShortUrlsEZ.com/payout-rates](https://ShortUrlsEZ.com/payout-rates)


== Installation ==

Upload .zip file to the `/wp-content/plugins/` directory and extract it
Active the plugin through the `Plugins` Menu in your Wordpress.
Good job you done, now Create your own short URLs.

= Options =

Once installed ShortUrlsEZ, Go to Settings > ShortUrlsEZ settings and configure your api.


== Features ==

= Allowed/No Allowed Domain =
If you have links with Domain.com then in the settings you put in the blacklist Domain.com and then the plugin will not short this domain.
